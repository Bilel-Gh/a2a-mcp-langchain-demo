Pillar 1: Agent Card

The business card for agents


What is it?

Before agents talk, they exchange cards. The card tells you everything you need to know.


Like a business card:

  • Who am I?
  • What can I do?
  • How to talk to me?
  • What format do I use?

‚Üí Now I know everything!

Agent Card { name: "Weather Agent" description: "Get weather forecasts" skills: [ "get_forecast", "get_temperature" ] inputs: text outputs: text }

Simple structure, clear information

Pillar 2: Messages

Quick conversations


What are Messages?

Messages are for quick, simple exchanges. Like a chat conversation.


Use for:

  • Fast questions
  • Simple requests
  • Direct responses
  • Real-time chat

Flow:
Request ‚Üí Response
Fast and simple!

Message { role: "user" or "agent" parts: [ { text: "What's the weather?" } ] message_id: "msg-123" }

Example:
"What's the weather in Paris?" ‚Üí "Sunny, 25¬∞C"

Pillar 3: Tasks

Long-running work


What are Tasks?

Tasks are for complex operations that take time. You can track progress.


Use for:

  • Trip planning
  • Data analysis
  • Report generation
  • Any work > 30 seconds

Lifecycle:
submitted ‚Üí in_progress ‚Üí completed

Task { status: "submitted" ‚Üì "in_progress" ‚Üì "completed" artifacts: [ results stored here as they are produced ] task_id: "task-456" }

Example:
"Plan a 7-day trip to Japan" ‚Üí Track progress ‚Üí Get complete itinerary
